The position

Not another wrapper around somebody else's runtime.

Forge is built for people who keep real models on disk and want a Mac app that understands the workflow: scan local folders, load and unload models deliberately, inspect the active prompt, tune sampling, and hand a local endpoint to tools that already speak OpenAI's API shape.

Why it matters

A Mac-native control surface for local inference.

01

MLX and GGUF in one workbench

Run MLX model trees and GGUF weights from a single native app, backed by Swift packages and an in-process LLM.swift backend.

02

Bring your own models

Point Forge at local model folders or an MLX-community cache. It discovers models recursively without shipping weights in the repo.

03

Operator-grade prompt control

Inspect named system prompt presets, chat-template behavior, thinking mode, sampling defaults, repetition penalty, and KV cache.

04

Local agent bridge

Expose a loopback OpenAI-compatible server so local tools can call your Mac model without wildcard CORS or a cloud relay.

05

MCP-aware, App Store-safe

Use HTTP/SSE MCP configuration for safe app workflows, with local stdio checks kept manual in developer builds.

06

Cloud optional

Anthropic, OpenAI, and OpenRouter keys are optional and stored with Keychain support. The core local path stays local.

How it works

A direct loop from disk to chat to tools.

  1. Point Forge at model folders.

    Add local directories or an MLX-community cache from the model browser.

  2. Load the model you actually want.

    Use explicit load, unload, active model, and stop controls.

  3. Tune the conversation surface.

    Set system prompts, template behavior, sampling, and cache settings.

  4. Talk to it from the app or a local tool.

    Use the native chat UI or the loopback OpenAI-compatible API.

Build

Public source. Native Mac build.

Forge is a SwiftPM package with executable products for the app and runtime tooling. It targets modern macOS on Apple Silicon and resolves dependencies from public git sources.

Open the repository 
git clone git@github.com:christopherbattlefrontlegal/swift-mlx-forge.git
cd swift-mlx-forge
swift build
swift run mlx-forge

Launch it
